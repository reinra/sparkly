# Future ideas

Here's some ideas for evolving the project:

- Add more effects
- Add support for sending mulitple movies and on-device playlists
- Add support for searching for effects by different criteria
- Add support for custom favorite playlists
- Improve support for seamless loops for more effects
- Add support for **virtual devices**
  - Rely only on the Live preview in the browser
  - Let user choose number of LEDs when adding (creating) the device
  - This enables to demo the app to any user without requiring a physical Twinkly device
  - It simplifies adding non-Twinkly vendor support in the future
- Enable special effect support for positioning LEDs in a [curtain](https://twinkly.com/en-eu/products/curtain)
  - Instead of generic 2D area, treat it as a dinstint number of LED strips
  - Experiment with new effects
- Add support for device groups, spanning same effect on multiple devices
- Experiment with Color organ / Music visualization
- Add mapping editor for fine-tuning it
